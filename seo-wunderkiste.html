Warum 14 Plugins installieren, wenn eines alles erledigt? Die Wunderkiste ist so programmiert, dass sie standardmaessig nichts tut. Du entscheidest unter Einstellungen, welche der 14 Superkraefte deine Seite braucht.

  • 14-in-1: Ersetzt diverse Einzel-Plugins fuer Performance und Sicherheit.
  • Null Ballast: Funktionen werden nur geladen, wenn du sie aktivierst.
  • Hand-Code Qualitaet: Maximale Effizienz ohne aufgeblaehte Bibliotheken.

Die 14 Module im Detail

Modularitaet als Prinzip:

Jede Funktion ist als eigenstaendiges Modul programmiert. Standardmaessig ist alles deaktiviert. Du entscheidest, was laeuft. Kein Alles-oder-Nichts, sondern chirurgische Kontrolle ueber deinen Stack.

1. SEO Schema (JSON-LD)
Fuegt Beitraegen und Seiten ein Eingabefeld hinzu, um individuellen Schema.org Code (JSON-LD) direkt in den Head einzufuegen.
2. Bulk NoIndex Manager
Ermoeglicht das massenhafte Setzen und Entfernen von NoIndex fuer Seiten und Beitraege direkt aus der Uebersicht. Inklusive Status-Spalte.
3. SEO Zombie Killer
Leitet sinnlose Anhang-Seiten (Attachment Pages), die Google nicht mag, automatisch per 301-Redirect auf den zugehoerigen Beitrag um.
4. Conversion Tracker
Ermoeglicht GA4 und Google Ads Conversion-Tracking auf einzelnen Seiten (z.B. Danke-Seiten). Unterstuetzt separate IDs und Conversion Values.
5. Image Resizer (800px)
Fuegt einen Button hinzu, um riesige Bilder mit einem Klick auf webfreundliche 800px herunterzuskalieren.
6. Upload Cleaner
Bereinigt Dateinamen automatisch beim Upload. Aus Mein Foto_Uebersicht.JPG wird automatisch mein-foto-uebersicht.jpg.
7. Zero-Click Image SEO
Generiert aus dem Dateinamen automatisch den Bild-Titel und den Alt-Text. mein-projekt.jpg erzeugt den Alt-Text Mein Projekt.
8. Media Inspector
Zeigt in der Medienuebersicht zwei neue Spalten an: Dateigroesse (KB/MB) und exakte Pixel-Abmessungen.
9. SVG Upload Support
Erlaubt das Hochladen von SVG-Dateien in die Mediathek, was WordPress standardmaessig aus Sicherheitsgruenden blockiert.
10. Emoji Bloat Remover
Entfernt das unnoetige JavaScript und CSS fuer WordPress-Emojis, um die Ladezeit messbar zu verbessern.
11. XML-RPC Blocker
Deaktiviert die XML-RPC Schnittstelle. Dies schuetzt deine Seite effektiv vor Brute-Force-Angriffen und unnuetiger Serverlast.
12. Login Tuersteher
Schuetzt deinen Admin-Bereich. Die Login-Seite ist nur noch erreichbar, wenn ein geheimer Parameter (z.B. ?hintereingang) an die URL angehaengt wird.
13. Comment Blocker
Deaktiviert Kommentare global auf der gesamten Website, entfernt Kommentar-Menues und schliesst bestehende Diskussionen.
14. ID Column Display
Zeigt die ID von Beitraegen, Seiten und Medien in allen Uebersichten an. Ein Klick auf die ID kopiert sie in die Zwischenablage.
